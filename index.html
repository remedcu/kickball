Terrible Kickball Rules

Lazy & Lousy Version

Be nice

Be fair

We believe in an honour system

Try to enjoy the game

Kicker Rules

Each kicker can have 3 chances to kick until they make a valid kick

A valid kick is any kick that has the ball's first touch inside the pitch and outside the small area near the kick zone

A rolling ball after a kick is considered invalid if it goes outside directly from near the kick zone

After a valid kick, if the kicker didn't run to first base or further, then the person is out

Running Rules

Only a single person should stay in one base

If there are two people or more on the same base, then we use the FIFO (First in First Out) mechanism to call out. So, the last person who was in the base stays

If a person ran too fast, and reached the base, but lost balance or so while the ball touched some base, then it is nice not to call them out

Pitcher Rules

If the ball is too bouncy or not coming at you, the kicker can leave the ball (and it would be counted as an invalid throw)

The pitcher is asked to pitch the ball rolling, not bouncing

Catching Rules

If the ball is caught directly, the kicker is out. The ball then needs to be thrown to the base where a runner is running to mark them out as well

All the runner who ran during the catch has to go back to the base before running again

Out Rules

The fielder should have the ball in any of the bases to mark the person running towards it out

The person should have the ball in hand while touching the base to call a person out

The person should say the word "out" to let other people know when he has the ball and is touching the base

The ball can be used to touch the running player to mark them out

Throwing the ball to the runner is not encouraged, as we are all beginners, and it does not mark them out
